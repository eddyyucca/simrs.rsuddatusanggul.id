Get started

Selamat datang di FKTP Khanza JKN Mobile API.
Layanan ini diperuntukkan untuk bridging antara server SIMRS Khanza dengan server JKN Mobile BPJS.

API Endpoint

{?= url('jknmobilefktp') ?}

Untuk menggunakan API ini, dibutuhkan username dan password untuk generate API key dalam bentuk token. Silahkan hubungi kami untuk mendapatkan username dan password.

get token

Untuk mendapatkan token, Anda perlu melakukan panggilan POST ke url berikut: :
{?= url('jknmobilefktp/auth') ?}


CATALOG

Field Description
Nama Fitur Get Token
Keterangan Mengambil token
URL {?= url('jknmobilefktp/auth') ?}
Contoh
Method GET
Header x-username: {user akses}
x-password: {password akses}

REQUEST PARAMETERS

Field Type Description 


Response :

{
    "response": {
            "token" : "1231242353534645645"
    },
    "metadata": {
        "message": "Ok",
        "code": 200
    }
}

RESPONSE PARAMETERS

Response Type Description
token String Kode acak hasil query.
Metadata Type Description
message String Status
code String Kode status

get status antrian

Untuk mendapatkan antrian, Anda perlu melakukan panggilan POST ke url berikut: :
{?= url('jknmobilefktp/antrean/status/[kode_poli]/[tanggalperiksa]') ?}


CATALOG

Field Description
Nama Fitur Get No Antrian
Keterangan Mengambil nomor antrian
URL {?= url('jknmobilefktp/antrean/status/[kode_poli]/[tanggalperiksa]')?}
{?= url('jknmobilefktp/antrean/status/001/2020-01-28')?}
Contoh
Method GET
Security TOKEN
Header x-username: {user akses}
x-password: {password akses}

REQUEST PARAMETERS

Field Type Description 


Response :

{
    "response": {
        "namapoli" : "Poli Umum",
        "totalantrean" : "25",
        "sisaantrean" : "4",
        "antreanpanggil" : "021",
        "keterangan" : ""
    },
    "metadata": {
        "message": "Ok",
        "code": 200
    }
}

RESPONSE PARAMETERS

Response Type Description
namapoli String Nama Poli FKTP.
totalantrean String Total antrian pada Poliklinik.
sisaantrean String Sisa antrian yang belum dilayani.
antreanpanggil String Antrian dalam pelayanan.
keterangan String Keterangan untuk pasien.
Metadata Type Description
message String Status
code String Kode status
